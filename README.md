# nodejs-weather

### Instructions

####  Download the code
```console
git clone https://github.com/joseanfernandez/nodejs-weather
```

####  Install dependencies
```console
cd nodejs-weather
npm install
```

#### How to use it
```console
npm start
```
Add 2 parameters in the url, city01 and city02.

[Try me!](http://localhost:8080?city01=Malaga&city02=London)
